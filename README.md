# SOS App

Приложение для Android: тройное нажатие кнопки громкости (вверх или вниз)
отправляет SMS доверенному контакту с текстом сообщения, временем и
ссылкой на текущую геолокацию (Google Maps).

## Как собрать

1. Установите Android Studio (последняя стабильная версия).
2. Откройте папку `SOSApp` как проект (File -> Open).
3. Если Android Studio предложит создать Gradle Wrapper — согласитесь,
   дайте проекту синхронизироваться (Sync Project with Gradle Files).
4. Подключите телефон по USB (включите режим разработчика и USB-debugging)
   или используйте эмулятор — но эмулятор не умеет отправлять реальные SMS.
5. Нажмите Run ▶ — приложение установится на телефон.

## Как настроить на телефоне

1. Откройте приложение, введите номер доверенного контакта и текст
   сообщения, нажмите «Сохранить».
2. Нажмите «Запросить разрешения» и разрешите доступ к геолокации и SMS.
3. Нажмите «Включить службу специальных возможностей», найдите в списке
   «SOS App» и включите переключатель.
4. Готово. Тройное быстрое нажатие громкости (в течение ~1.5 сек) отправит
   SOS-сообщение.

## Важные ограничения, о которых стоит знать

- **Экран заблокирован / выключен.** На многих устройствах (особенно
  Samsung, Xiaomi, Huawei и др.) службы специальных возможностей могут
  не получать нажатия кнопок при выключенном экране из-за агрессивной
  экономии батареи производителя. Может потребоваться вручную отключить
  оптимизацию батареи для этого приложения (Настройки -> Батарея ->
  без ограничений).
- **SMS, а не интернет-сообщение.** Используется обычный SmsManager —
  значит на SIM-карте должны быть деньги/баланс для отправки SMS.
  При желании можно заменить отправку SMS на запрос к мессенджер-API
  (Telegram Bot API, WhatsApp Business API и т.п.) — для этого нужен
  интернет и токен бота.
- **Ложные срабатывания.** Тройное нажатие громкости может случайно
  происходить в кармане. Можно увеличить REQUIRED_PRESSES или добавить
  подтверждение (например, вибрацию + 3-секундное окно отмены) в
  `VolumeKeyAccessibilityService.kt`.
- **Политики Google Play.** Для публикации в Play Store сервисы
  специальных возможностей требуют отдельного декларирования и проверки
  Google. Для личного использования (установка через Android Studio /
  APK напрямую) это не требуется.

## Структура проекта

- `MainActivity.kt` — экран настроек (контакт, сообщение, разрешения).
- `VolumeKeyAccessibilityService.kt` — основная логика: слушает кнопки
  громкости, получает геолокацию, отправляет SMS.
- `PrefsHelper.kt` — хранение номера и текста сообщения.
- `res/xml/accessibility_service_config.xml` — конфигурация службы,
  включающая перехват клавиш (`flagRequestFilterKeyEvents`).
